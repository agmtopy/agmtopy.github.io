环境准备

  • WSL
    wsl2搭配Ubuntu 18.04使用,这一项不是必须的

  • Clion
    JetBrains的C/C++ IDE,这一项也不是必须的

  • 源代码
    https://github.com/redis/redis

构建过程

  • 按照Linux编译工具
apt update
apt install git  cmake make gcc g++ gdb
  • 构建release.h
#授权
chmod +x mkreleasehdr.sh
#执行make
make install

以上两个步骤就完成redis源代码的编译,接下来让我们把源代码导入CLion中进行debug
